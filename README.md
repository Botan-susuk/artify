# 🎨 Artify - Art Marketplace Web App

## 🚀 Features

- 🔐 User Authentication (Register / Login / Logout)
- 👤 User Profile (Update fullname & avatar)
- 🎨 Artwork Management
  - Add new artwork
  - View gallery
  - View artwork details
- 🛒 Shopping Cart
  - Add to cart
  - Remove from cart
- 📦 Order System
  - Place orders
  - View order history
- 🌙 Dark Purple UI Theme (modern & clean)
